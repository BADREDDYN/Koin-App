# Koin-App
Sample app to try Koin library with ViewModel.

> Add the dependencies:

  ```gradle
  ...
	dependencies {
    ...
    // Koin
    implementation "io.insert-koin:koin-android:3.2.0-beta-1"
	}
  ```
> Go to AndroidManifest and add:
  ```AndroidManifest
  <application
        android:name=".di.MyApp"
        ...
  </application>
  ```
